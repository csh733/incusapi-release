# IncusAPI

Incus web management system with API. Single binary, installs directly on the host.

## One-Click Install

Run on a fresh **Debian/Ubuntu** or **RHEL/CentOS** machine:

```bash
curl -sSL https://raw.githubusercontent.com/csh733/incusapi-release/master/install.sh -o /tmp/incusapi-install.sh && sudo bash /tmp/incusapi-install.sh
```

This automatically installs:
- System dependencies (curl, jq)
- Incus (via Zabbly repo)
- Incus default configuration (storage pool + network)
- IncusAPI binary + systemd service
- Firewall rules

After install, access the web UI at `http://SERVER_IP:8080`.

## Features

- Web dashboard (Vue 3 + Element Plus)
- Full Incus management — instances, networks, storage, images, profiles, projects
- WebSocket terminal (browser-based console)
- Resource monitoring with charts
- Billing system integration API
- JWT + API Key authentication
- Webhook notifications
- Auto-expiry management
- Single binary (~9MB compressed)

## Uninstall

```bash
curl -sSL https://raw.githubusercontent.com/csh733/incusapi-release/master/uninstall.sh -o /tmp/incusapi-uninstall.sh && sudo bash /tmp/incusapi-uninstall.sh
```

## Supported Platforms

| Architecture | Binary |
|---|---|
| x86_64 (amd64) | `incusapi-linux-amd64` |
| ARM64 (aarch64) | `incusapi-linux-arm64` |

## Configuration

After install, edit `/etc/incusapi/config.yaml`:

```yaml
server:
  host: "0.0.0.0"
  port: 8080
  secret: "<auto-generated>"

incus:
  socket: ""  # default: /var/lib/incus/unix.socket

branding:
  site_name: "IncusAPI"
  logo_url: ""
```

Then restart: `systemctl restart incusapi`
